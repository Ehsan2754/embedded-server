# ESP32server
A Server based on ESP32 serving BLE and Web over network for ZARNITZA company
# Compile options
  1. ESP32 dev module
  2. Upload speed 921600
  3. Flash configurations:    
      * Cpu speed 240 mhz
      * Flash speed 80 mhz
      * Flash mode qio
      * Flash size 4mb
      * Partition scheme default
